# Gorilla Order v1.0 Alpha

A structured Next.js version of Gorilla Order.

## Includes
- Multi-step sticker checkout
- Live pricing
- Shipping calculator
- Artwork upload UI
- Customer form
- Printavo quote creation
- Confirmation page with quote links

## Vercel Environment Variables
- PRINTAVO_EMAIL
- PRINTAVO_TOKEN
- PRINTAVO_TEST_CUSTOMER_ID

## Run
npm install
npm run dev

## Note
This alpha still uses the test customer ID. Do not use for production customer orders yet.
